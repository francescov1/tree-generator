# Tree Generator

This script generates a markdown representation of your current directory structure, resembeling the following:

├── index.js
├── tests
│ ├── moreTests
│ │ ├── test3.js
│ ├── test1.js
│ ├── test2.js
├── controllers
│ ├── controller1.js
├── models
│ ├── model1.js
└─└── model2.js

The structure is saved in `tree.md`.
